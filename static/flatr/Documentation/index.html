Template SetupTheme customization and highlights

Counter Setup

First, you need to open index.html file, scroll to bottom of it and locate this line: austDay = new Date(2013, 6 - 1, 23);

All you need to do is change the year (2013), change the month (6) and change the date (23).

For example, if your date is February 05, 2014, the code would look like this: austDay = new Date(2014, 2 - 1, 5);

Add your own Twitter account.

If you want to add your own twitter feed all you have to do is locate this line of code in the index.html file username: "avathemes", and replace the avathemes with your own username

Change Photo Backgrounds

If you're using the photo background version, you'll need to add your own photos. Photo placeholders can be located inside the images/background-slideshow folder and there you can see the size of the images we've used. However if you want to use other image size you can do that as well. We recomment any size between 1680x900 - 1680x1200.

If you're images are too large and if the site loading speed is low, please resize your images at jpegmini.com

If you want to add mroe than three images, you'll have to open the index.html file and modify the supersized settings on line 301:
image : 'images/background-slideshow/1.jpg', url : ''},
{image : 'images/background-slideshow/2.jpg', url : ''},
{image : 'images/background-slideshow/3.jpg', url : ''}

Change Video Background

If you're using the video background version, you'll have to convert your video to .mp4, .ogv (.ogg) and you'll need to prepare one background image (for tablets and smartphones)

In orer to make your video version work perfectly, please place your videos to video folder, open index.html and locate this code:

if (Modernizr.touch) {
BV.show('images/backgrounds/1.jpg');
} else {
BV.show('video/1.mp4', {altSource:'video/1.ogv'});
}

you just need to replace 1.mp4, 1.ogv and 1.jpg with your video title and that's it.
